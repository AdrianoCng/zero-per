# Zero Per

## Installation
First run `npm install` to install all the dependencies.

Run `npm run dev` to start the project

### Development

```HTML
<div class="wrapper">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>

  <div class="box"></div>
  <div class="box"></div>

```
Write your css in ***./sass/layout*** - use **_board.scss** to style the board and **_box.scss** to the boxes.
